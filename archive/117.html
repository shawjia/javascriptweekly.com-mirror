Event Based Programming: What Async Has Over Sync
One of JavaScript’s strengths is how it handles asynchronous (async for short) code. Rather than blocking the thread, async code gets pushed to an event queue that fires after all other code executes. It can, however, be difficult for beginners to follow async code. This article explains how it works.
